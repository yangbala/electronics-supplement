# 電子學補充教材網站：極點與零點頁面設計規格

**日期：** 2026-05-24  
**狀態：** 已確認，待實作

---

## 1. 專案概述

建立一個電子學補充教材靜態網站，第一個頁面主題為「極點與零點在左半平面與右半平面對系統的影響」。頁面以全互動拖曳式視覺化為核心，搭配教學文字說明，適合從初學到進階的混合程度大學生。

---

## 2. 技術選型

| 項目 | 選擇 | 理由 |
|------|------|------|
| 框架 | 無（Vanilla JS + ES Module） | 無需建置工具，開啟瀏覽器即可使用 |
| 視覺化 | D3.js v7（本機副本） | 最適合數學互動 SVG 視覺化 |
| 數值計算 | 純 JS 自製 | 步階響應（Euler 法）、Bode（虛軸掃描） |
| 樣式 | 純 CSS（無框架） | 深色工程主題，減少依賴 |
| 離線支援 | lib/ 存放 D3 本機副本 | 教室環境不一定有網路 |

## 4. 頁面結構（poles-zeros.html）

由上到下：

1. **導覽列** — 網站名稱 `⚡ 電子學補充教材` + 各單元連結（後續擴充）
2. **標題區** — 頁面主題標題 + 一句話說明
3. **概念說明區** — 四張折疊式說明卡（預設展開），含「載入範例」按鈕
4. **預設情境按鈕列** — 五個快速情境按鈕
5. **互動主區** — 左右分割（s 平面 50% ｜ 三圖 50%）
6. **深入說明區** — 可選讀的數學推導與電路範例

## 5. 互動主區設計

### 5.1 s 平面（左側 50%）

- **座標系：** SVG，橫軸 σ（實部）、縱軸 jω（虛部）
- **視覺區分：** 左半平面淡藍色背景、右半平面淡紅色背景
- **符號：** 極點用 × 符號（LHP=藍色、RHP=紅色）；零點用 ○（金黃色）
- **拖曳行為：**
  - 滑鼠按住拖曳移動極點/零點
  - 複數共軛對自動鏡像（拖曳其中一個時另一個同步對稱移動；工具列可解除共軛鎖定，使兩極點獨立）
  - 極點落入 RHP 時顯示紅色警示光暈動畫
- **新增/刪除：**
  - 點擊空白區域新增（依工具列目前模式：極點 or 零點）
  - 右鍵點擊刪除
  - 工具列切換「新增極點」/「新增零點」模式
- **事件：** 每次拖曳結束後發出 `onPoleZeroChange`，觸發三圖同步重算

### 5.2 右側三圖（垂直堆疊）

**① 步階響應圖（`step-response.js`）**
- D3 折線圖，x 軸時間（自動縮放），y 軸幅度
- 穩定系統：顯示終值虛線
- 不穩定系統（任一 RHP 極點）：y 軸截斷 + 顯示「⚠ 系統發散」
- 虛軸極點：等幅振盪，標示振盪頻率 ω₀

**② 波德圖（`bode-plot.js`）**
- 上下兩個子圖：增益（dB）+ 相位（度），共用對數頻率 x 軸
- 頻率範圍：0.01 ~ 1000 rad/s
- 極點/零點頻率位置用垂直虛線標示
- 滑鼠懸停顯示該頻率的增益/相位數值

**③ 系統資訊面板（`info-panel.js`）**
- 極點座標列表
- 零點座標列表
- 穩定性判斷：`✓ 穩定` / `✗ 不穩定` / `⚠ 臨界穩定`
- 阻尼比 ζ（共軛複數極點對）
- 自然頻率 ωₙ
- 直流增益 H(0)

## 6. 傳遞函數數值計算（`transfer-function.js`）

傳遞函數以極零點形式定義：

```
H(s) = K · ∏(s − zᵢ) / ∏(s − pⱼ)
```

預設 K = 1；工具列提供增益滑桿（0.1 ~ 10，對數刻度）。

- **步階響應：** 對 `H(s)/s` 做數值反拉普拉斯，使用 Euler 法離散化，取樣 1000 點，時間範圍自動根據最慢極點的時間常數決定
- **波德圖：** 沿虛軸掃描 `s = jω`（對數間隔 500 點），計算 `20·log₁₀|H(jω)|`（dB）與 `arg(H(jω))`（度）
- **複數運算：** 以 `{re, im}` 物件表示複數，實作加、乘、除、模、幅角等基本運算

## 7. 概念說明卡（四張）

| 卡片主題 | 核心說明 | 範例情境 |
|----------|----------|----------|
| LHP 極點（σ < 0） | 指數衰減，系統穩定；越靠左衰減越快 | 一階穩定、二階欠阻尼 |
| RHP 極點（σ > 0） | 指數增長，系統不穩定；任一 RHP 極點即發散 | 一階不穩定 |
| 虛軸極點（σ = 0） | 等幅振盪，臨界穩定，頻率 = |ω| | 臨界穩定振盪 |
| RHP 零點 | Non-minimum phase：步階響應初始方向反向，相位滯後超出增益預期 | 右半平面零點 |

每張卡片包含：文字說明 + 對應數學式 + 「點此載入範例」按鈕

## 8. 預設情境按鈕（五個）

| 按鈕標籤 | 極點/零點設定 | 用途 |
|----------|--------------|------|
| 一階穩定 | p = −2 | 最基本穩定系統 |
| 二階欠阻尼 | p = −1 ± 3j（共軛對） | 展示振盪收斂 |
| 臨界穩定 | p = ±3j（純虛） | 展示等幅振盪 |
| 一階不穩定 | p = +1 | 展示 RHP 極點發散 |
| RHP 零點 | p = −2，z = +1 | 展示 Non-minimum phase |
